Readme for PicMcuCommII  05.05.2005

To communicate with your PIC microcontroller using the PicMcuComm.exe:

1. Copy all files in this directory to a convenient location on your PC.
2. For convenience you may also create a shortcut to the PicMcuCommII.exe and
   place it on your desktop.
3. Double click on the PicMcuCommII.exe file name or shortcut.
